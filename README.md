# SoftIto Backend Development Portfolio

Welcome to my backend development portfolio! This repository serves as a centralized hub for the projects I developed during the **SoftIto Backend Developer Training Program**, organized in collaboration with the **İstanbul Ticaret Odası (İTO)**. 

---

## About Me

I am a **Mathematics Engineering** graduate from **Yıldız Technical University** (100% English curriculum). With a strong foundational background in analytical problem-solving and mathematical logic, I transitioned into software engineering to focus on building robust, scalable, and efficient backend systems. 

Through intensive training and hands-on project development, I have honed my skills in the .NET ecosystem, data architecture, and modern backend practices.
